## 1. Reels Script Templates

### Template A: ASMR Rót Bia (15-30s)

```
[0-3s] HOOK: Close-up nắp bia → tiếng "psshh" mở nắp
[3-10s] BUILD: Rót chậm vào ly, camera nghiêng 45°, bọt dâng
[10-20s] PEAK: Zoom vào bọt trắng xốp, ánh sáng ấm
[20-25s] REVEAL: Zoom out → full setup: bia + đồ ăn + nến
[25-30s] END CARD: Text overlay: Tên bia + logo nhỏ

CAPTION: "[Tên bia] — Nghe thử xem 🎧
[Fact 1 câu về loại bia này]
📌 Save để cuối tuần thử nhé"

AUDIO: ASMR sound only (không nhạc nền)
SUBTITLE: "Âm thanh của [tên bia] — brewed since [năm]"
```

### Template B: POV (15-20s)

```
[0-2s] TEXT: "POV: [tình huống relatable]"
[2-8s] SCENE: Camera subjective, di chuyển tự nhiên
[8-15s] REVEAL: Mở bia / rót bia / setup bàn
[15-20s] END: Text "Bia Thầy Tu — [tagline ngắn]"

CAPTION: "[Câu hook liên quan đến POV]
Ai giống mình? 🙋‍♂️
#POV #BiaThayTu #BiaĐức"

AUDIO: Trending sound hoặc nhạc chill
```

### Template C: Fact Bất Ngờ (20-30s)

```
[0-3s] TEXT HOOK: "[Con số bất ngờ] 🤯"
[3-10s] EXPLAIN: Voiceover giải thích fact
[10-20s] VISUAL: B-roll bia, nhà máy, nguyên liệu
[20-30s] TWIST: "Và đây là điều ít người biết..."
         REVEAL thêm 1 fact nữa

CAPTION: "[Hook question]
[Expand fact 2-3 câu]
Bạn biết fact nào về bia Đức? Comment 👇"
```

## 2. Carousel Templates

### Template A: Guide / How-to (5-7 slides)

```
SLIDE 1 — HOOK
[Visual: Ảnh đẹp, premium]
Title: "Cách [hành động] — [benefit]"
Subtitle: "[số] bước đơn giản"

SLIDE 2-5 — CONTENT
[Mỗi slide = 1 bước/1 ý]
Layout: Icon/số + Tiêu đề ngắn + 1-2 câu giải thích
Visual: Ảnh minh hoạ hoặc icon

SLIDE 6 — SUMMARY
[Tóm tắt dạng checklist visual]

SLIDE 7 — CTA
"Save để dùng sau 📌"
"Share cho bạn bè sành bia 🍻"
Logo Bia Thầy Tu nhỏ góc

DESIGN:
- Background: Navy dark (#0a1628)
- Text: Gold (#d4a853) cho heading, White cho body
- Font: Sans-serif, clean
- Consistency: Cùng layout mỗi slide
```

### Template B: So Sánh (4-6 slides)

```
SLIDE 1 — HOOK
Title: "[A] vs [B] vs [C]"
Subtitle: "Bạn thuộc team nào?"
Visual: 3 sản phẩm side-by-side

SLIDE 2 — ITEM A
[Tên + Hình + 3 đặc điểm key]
"Best for: [dịp/mood]"

SLIDE 3 — ITEM B
[Tương tự slide 2]

SLIDE 4 — ITEM C
[Tương tự slide 2]

SLIDE 5 — COMPARISON TABLE
[Bảng so sánh visual: Hương, Vị, Độ cồn, Dịp uống]

SLIDE 6 — CTA
"Bạn team nào? Comment bên dưới 👇"
```

### Template C: Bảng Pairing (5-8 slides)

```
SLIDE 1 — HOOK
Title: "Bia nào × Món nào?"
Subtitle: "Menu chuẩn cho [số] dịp"

SLIDE 2-7 — PAIRINGS
[Mỗi slide = 1 cặp bia + món]
Layout: Ảnh bia bên trái + Ảnh món bên phải
Text: Tên bia • Tên món • Tại sao hợp (1 câu)

SLIDE 8 — CTA
"Save menu này cho lần tới 📌"
"Bạn đã thử cặp nào? Comment cho mình biết 🍽️🍺"
```

## 3. Photo + Caption Templates

### Template A: Storytelling (Brand story)

```
PHOTO: [Ảnh atmospheric — setup bàn bia, ánh sáng ấm, cinematic]

CAPTION:
"[HOOK — 1 câu gây cảm xúc/tò mò]

[ORIGIN — 2-3 câu về nguồn gốc/hành trình]

[DETAIL — 1-2 câu về nguyên liệu/quy trình đặc biệt]

[PERSONAL — 1 câu cảm nhận cá nhân/quan điểm]

[CTA — Câu hỏi mở]

#BiaThayTu #BiaĐức #CraftBeer #PremiumBeer #[LoạiBia]"
```

### Template B: Education (Kiến thức)

```
PHOTO: [Ảnh close-up sản phẩm hoặc infographic mini]

CAPTION:
"[HOOK — Fact bất ngờ hoặc myth buster]

📌 [Kiến thức 1 — 1-2 câu]

📌 [Kiến thức 2 — 1-2 câu]

📌 [Kiến thức 3 — 1-2 câu]

[TAKEAWAY — 1 câu kết luận/insight]

[CTA — Câu hỏi liên quan]

#BiaĐức #KiếnThứcBia #BeerFacts"
```

### Template C: Social Proof (Review/Testimonial)

```
PHOTO: [Ảnh khách hàng thật hoặc sản phẩm in-context]

CAPTION:
"[HOOK — Quote từ khách hoặc reaction]

[CONTEXT — Tình huống: ai, ở đâu, dịp gì]

[REACTION — Cảm nhận thật, chi tiết cụ thể]

[BRIDGE — Liên kết với sản phẩm/trải nghiệm]

[CTA — 'Bạn đã thử chưa? Share trải nghiệm của bạn 👇']

#BiaThayTu #Review #CustomerLove"
```

## 5. Batch Content Production Workflow

```
Bước 1: PLAN (30 phút/tuần)
  → Chọn 7 topics theo content mix 40/30/20/10
  → Assign format cho từng topic
  → Viết outline hooks

Bước 2: CREATE (2-3 giờ/tuần)
  → Batch quay Reels (quay 3 video 1 lần)
  → Batch thiết kế Carousel (dùng template)
  → Viết caption theo cấu trúc 4 phần

Bước 3: REVIEW
  → Chạy pre-publish checklist
  → Humanize content (skill: vietnamese-content-humanizer)
  → Proofread + kiểm tra brand consistency

Bước 4: SCHEDULE
  → Lên lịch theo khung giờ vàng
  → Setup alarm reply comment

Bước 5: ENGAGE (30 phút/ngày)
  → Reply tất cả comment trong 30 phút đầu
  → Pin comment hay nhất
  → Cross-pollinate giữa các page
```
