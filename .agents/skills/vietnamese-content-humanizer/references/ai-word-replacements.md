# Vietnamese AI Word Replacements

Bảng thay thế từ/cụm từ "AI quá" → từ tự nhiên tiếng Việt. Dùng khi review output.

---

## Từ nối (Connectors)

| ❌ AI dùng | ✅ Người Việt nói | Ghi chú |
|-----------|------------------|---------|
| Tuy nhiên | Nhưng mà / Cơ mà / Thế nhưng | Phổ biến nhất |
| Do đó | Thế nên / Nên là / Cho nên | |
| Ngoài ra | Với lại / Rồi còn / À mà | |
| Sau đó | Xong rồi / Rồi / Thế là / Xong sau đấy | |
| Điều đáng chú ý là | Cái hay ở đây là / Hay ở chỗ | |
| Có thể kết luận rằng | Chốt lại thì / Nói chung là / Tóm lại | |
| Mặc dù vậy | Dù vậy thì / Kệ, dù gì thì | |
| Trước hết | Đầu tiên là / Cái đầu tiên | |
| Cuối cùng | Rồi cuối cùng thì / Sau hết thì | |
| Bên cạnh đó | Với lại / Mà còn | |
| Đặc biệt | Nhất là / Đặc biệt là (chấp nhận được) | |
| Trên thực tế | Thật ra thì / Nói thật thì | |
| Nhìn chung | Nói chung thì / Nhìn lại thì | |
| Theo đó | Thế là / Rồi thì | |
| Vì vậy | Nên / Thế nên / Cho nên | |

## Từ đệm tự nhiên (Filler Words)

Rắc đều khắp bài, đặc biệt đoạn mở đầu.

### Đầu câu
- Tự dưng, Nói thật là, Ừ thì, Thú thật, Nói nhanh thì
- Kiểu như, Đại khái, Nói gì thì nói

### Giữa câu  
- kiểu, đại khái, tầm, cũng, thì, mà, cơ, á

### Cuối câu
- luôn á, thôi á, đó, nha, nghen, nè, đấy, cơ, lắm

### Cảm thán
- Ơ, Ủa, Trời ơi, Wow, Á đù, Chà, Ôi

## Blacklist — Từ AI hay dùng (TRÁNH)

### Từ mở bài sáo rỗng
| ❌ TRÁNH | Lý do |
|---------|-------|
| Trong thế giới hiện đại ngày nay | Mở bài kinh điển của AI |
| Bạn đã bao giờ tự hỏi... | Câu hỏi tu từ nhàm |
| Chào mừng bạn đến với... | Quảng cáo trắng trợn |
| Hãy cùng khám phá... | Sáo rỗng |
| Không thể phủ nhận rằng... | Văn báo cáo |

### Từ trang trọng quá mức
| ❌ TRÁNH | ✅ Thay bằng |
|---------|------------|
| Thưởng thức | Uống / Thử / Nhâm nhi (dùng tối đa 1 lần/bài) |
| Tuyệt vời / Tuyệt hảo | Ngon thiệt / Đỉnh / Hay lắm |
| Nâng tầm trải nghiệm | Thử cái mới / Khác hẳn |
| Hành trình vị giác | Uống thử là biết |
| Sản phẩm chất lượng cao | Đồ ngon / Hàng xịn |
| Mang đến cho bạn | Giới thiệu / Rủ thử |
| Đẳng cấp | Chất lượng / Xịn (tuỳ context) |
| Sự kết hợp hoàn hảo | Hợp lắm / Đi chung là chuẩn |
| Trải nghiệm đỉnh cao | Thử rồi sẽ biết |

### Từ chuyên ngành (Technical Jargon)
| ❌ TRÁNH | ✅ Thay bằng |
|---------|------------|
| Khoảng cách kỳ vọng | Rẻ ngoài sức tưởng tượng / Được nhiều hơn tưởng |
| Giá trị vượt kỳ vọng | Được nhiều hơn mình nghĩ |
| Tối ưu hoá | Làm cho tốt hơn |
| Insight | Điều hay ho phát hiện ra |
| Engagement | Lượng tương tác |
| Narrative arc | Mạch chuyện |
| Brand awareness | Nhiều người biết đến |
| Hiệu ứng leo thang | (Nói cụ thể hiệu ứng gì) |
| Pain point | Vấn đề đang gặp |
| Target audience | Nhóm khách hàng |
| Authentic | Thật / Chân thật / Mộc mạc |
| Viral content | Content hot / Bài đang hot |

## Cụm từ thêm cảm xúc

### Ngạc nhiên
- "Mình xem xong mà sốc luôn"
- "Ai ngờ..."
- "Hoá ra..."
- "Nói thật là mình không nghĩ tới"
- "Mặt mình kiểu ?? :D ??"

### Thích thú
- "Ngon lắm luôn"
- "Đỉnh thật sự"
- "Cái này phải thử"
- "Ghiền luôn từ lần đầu"

### Tò mò
- "Mình cũng tò mò nên thử luôn"
- "Ban đầu mình cứ tưởng..."
- "Ngồi nghiên cứu 1 hồi thì mới vỡ lẽ"

### Hài hước nhẹ
- "Khổ nỗi..."
- "Buồn cười ghê"
- "Nói nhanh cho nó vuông"
